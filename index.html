Creating the network

To start creating the network, first we need to define which were going to be the nodes and which would be the links between them. In our case, we decided that the ingredients of each recipe would be the nodes of the network. The link on the other hand would be the recipes they share. This means that, if two ingredients appear in the same recipe they would linked in the network. Moreover, we decided to create the network as a weighted network, where the weight of the links would be the amount of recipes the ingredients sare. In total we 39774 recipes and 1817 ingredients. With this, we created a network with 1817 nodes and 41325 links. As mentioned before it is a weighted and undirected graph.

The next step was to assign each ingredient to a cuisine. Up to this point, what we have in the network is a mess of ingredients mixed up with links conecting them. Since our goals are to compare this network to reality we need to sign each ingredient to a different cuisine.

From our data, we can extract that we have 20 different types of cuisines: irish, mexican, chinese, filipino, vietnamese, moroccan, brazilian, japanese, british, greek, indian, jamaican, french, spanish, russian, cajun creole, thai, southern US, korean and italian. To assign each ingredient to its cuisine we need to check in which recipes the ingredient appears and add it to the cuisine that recipe belongs. However, many ingredients aren't unique to a cuisine. To solve this issue we aproached the solution in two ways and after seeing the results we selected the one we thought was most representative of reality. Here we will explain the methos we finally used, but the other can be found in the notebook.

The method we chose assigned each ingredient to the cuisine in which it appeared most. Let's see this with an example from the notebook:

Introduction to sentiment analysis

In this section we are going to make some sentiment analysis of the different cuisines with Twitter. We have gathered more than 1000 tweets from each cuisine, searching with the hashtag "#spanishcuisine" for example for spanish cuisine suing the Twitter API. After cleaning all the tweets, we took the text from each one and analyzed the sentiment based on the LabMt wordlist, that contains an average sentiment for a number of words. This are the results for the analysis:

As we can see from the graph on the left, all the cuisines are quite close in the range of 5 to 6 average sentiment. However, if we adjust the range of the axis like the graph on the right, we can see the differences a bit more clearly. The fist thing to notice is that the sentiment related to american a british cuisines is significantly lower than the rest. In a second level, with an average sentiment level we find mexican, italian, greek or japanese cuisines , and above those with the best sentiment are chinese, cajun or vietnamese cuisines. We find quite interesting that cuisines with usually "bad reputation" like the british and american ones are the ones with the lowest rating. On the ohter hand asian quisines have good reputation in the western countries, so that might be a reason why they are rated so high.
